## Installation
Run this in your project and it'll save to your dependency lists
```bash
composer require cali/socialite
```

## Supported OAuth Services
- Disqus
- Douban
- Dribbble
- Facebook
- Github
- Google
- LinkedIn
- QQ
- Slack
- SoundCloud
- Spotify
- Weibo
- YouTube
